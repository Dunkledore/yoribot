Yori Bot

Yori-Bot is a highly customisable discord bot with features such welcome messages, custom user profiles and social media integration.

Yori has a very active developement team and, within reasonable limits, we will help you with all aspects of the bot. Yori also boasts a full web dashboard for moderation settings along with standard user features such us profile and cookies.
