About Yori

Yori is an advanced Discord bot being developed by a team of Discord bot enthusiasts. We all really love innovation and wanted to bring a robust set of commands and features to our users. Many of the features we are able to offer are thanks to other developers not even on our team. Because Yori is based on discord.py we have been able to migrate quite a few commands that our users have grown familiar with from the very popular Red Discord bot. To be clear, Yori is not a clone of Red and actually uses a different version of discord.py so when you see familiar commands and features they have been completely rewritten to function on Yori. The bot itself was based on the rewrite of rdanny developed by Rapptz (the same person responsible for discord.py). While we have utilized some of the base structure of robodanny the bot has been very heavily modified to meet our unique needs.

Code Credits

In instances where we have utilized an existing command or feature we would like to give credit to the original authors for their original work. Again, in all cases the code being used for Yori has been completely rewritten and are now 100% different than when originall introduced to the code community, however we like to give credit where credit is due.
