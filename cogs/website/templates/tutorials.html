Getting Started

For simplicity, we will assume the commands prefix of * for these tutorials. If you have a different prefix replace the * at the start of a command with the prefix you use.

Permissions

Guild Owner - This role has the highet power and will be able to execute all commands. Nothing is required for this apart from you being the owner of the guild. Steps required for this:
Be the guild owner
Adminis - This role has access to almost all of the commands except a very select few where account details are involved. Steps required for this: 
					
						Have the administrator permission in the guild
Mods - This role has access to all of the moderation commands in the server. This role must be set. Steps required for this: 
					
						*modrole @role

Greeter/SpecialRoles/Tweeter - These are roles which grant access to very specific function. See each relevant section for more information.

These roles work on an inheritcance basis. So Each role automatically has all the functionality as the roles below it.
